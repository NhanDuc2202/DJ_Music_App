# AI_CHAT_LOG.md

## 1. What I asked AI to help with
I used AI as a support tool for:
- planning the project structure
- organizing the app using a simple MVVM approach
- generating and refining Jetpack Compose UI code
- improving reusable components

## 2. Key prompts used
Examples of prompts I used:
- Help me structure a Kotlin Jetpack Compose Android project for a mobile UI test
- Suggest a clean folder structure using a lightweight MVVM approach
- Help me build a reusable banner card and grid menu card in Jetpack Compose
- Help me create a Settings screen with reusable rows and switches

## 3. What I accepted
I accepted AI support for:
- initial project structure suggestions
- reusable Compose component ideas
- layout refinement suggestions

## 4. What I rejected
I rejected or simplified suggestions that:
- introduced overly complex architecture for the scope of the test
- added unnecessary persistence or backend-like logic
- made the implementation heavier than required for a UI-focused technical test

## 5. What I modified
After receiving AI suggestions, I manually adjusted:
- spacing, padding, and alignment
- font usage and typography
- banner layout and image overflow behavior
- gradient border behavior
- component organization to better fit the provided design

## 6. How I verified the output
I verified the output by:
- comparing the UI visually against the provided design images
- running the app on emulator
- checking navigation between screens
- testing basic user interactions such as clicking cards and toggling switches
- refining layout details manually where needed
